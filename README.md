
# Simuladores e Estudos do Movimento Browniano 🧮

Este projeto consiste em simuladores em Python que modelam o movimento Browniano e geram arquivos e gráficos detalhados para estudos aprofundados deste fenômeno. Estamos conduzindo uma análise multifocal para ampliar a compreensão das diversas áreas de aplicação do movimento estudado. Vale ressaltar que, futuramente, pretendemos concentrar nossos esforços em uma única área de estudo, visando um aprofundamento ainda maior nos aspectos específicos do movimento.

## Funcionalidades 🛠️

- 1 - Modelo Base: DLA aplicando o movimento browniano que junto com o gráfico da evolução de energia por ponto permitiu dar um pontapé inicial na pesquisa

- 2 - Modelo Plantae: DLA aplicando o movimento browniano agoraa com seus pontos iniciais no formato de folhas de monocotiledôneas e dicotiledôneas para analisarmos a possibilidade da relação entre o movimento browniano e o crescimento biológico e focado em eficiência das plantas

- 3 - Modelo com Peso: Simulador de interação de partículas carregadas em um espaço bidimensional, utilizando forças de atração e repulsão baseadas na Lei de Coulomb, além de movimento browniano e fusão por colisão. As partículas têm massa, carga e raio realistas, inclui também versão simplificada que melhor mostra os efeitos do movimento browniano.


## Rodando localmente 🔋

Clone o projeto

```bash
  git clone https://github.com/EduLoboM/Movimento-Browniano
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  pip install matplotlib
```

Inicie o simulador

```bash
  py MoviBrown.py
```
